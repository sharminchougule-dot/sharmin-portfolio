## Image Requirements

- **Dimensions:** 800x400 pixels (or 2:1 aspect ratio)
- **Format:** JPEG (.jpg)
- **File Names:** 
  - publication-1.jpg
  - publication-2.jpg
  - publication-3.jpg
  - publication-4.jpg
  - publication-5.jpg
  - publication-6.jpg
- **Location:** Save all images in `static/images/` folder

## How to Download

1. Visit any of the suggested sites above
2. Search using the keywords provided
3. Download images (all these sites offer free downloads)
4. Resize images to 800x400 pixels if needed (you can use online tools like https://www.iloveimg.com/resize-image)
5. Rename and save to `static/images/` folder

## Alternative: Use the Image Generator Script

If you want to generate placeholder images programmatically:
1. Install Pillow: `python -m pip install Pillow`
2. Run: `python generate_publication_images.py`
3. This will create 6 themed placeholder images automatically
